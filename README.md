hxPixel
=======

Pixel Image Library for Haxe3

wip

Decoder
-------
* PngDecoder
* GifDecoder
* EdgDecoder (supported Edge1, Edge2)
* GalDecoder (supported Gale106 / GaleX200. bpp(bitDepth) <= 8)
* PsdDecoder (Incomplete)

Encoder
-------
* PngEncoder (Incomplete)
* GifEncoder
* PsdEncoder (Incomplete)
